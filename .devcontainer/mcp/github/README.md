# GitHub MCP Server

Official Go-based GitHub MCP server.

## Purpose
Repository management, issues, PRs, actions, security.

## Environment
- GITHUB_TOKEN (required)
- GITHUB_HOST: https://github.com

## Network
- IP: 172.20.0.10
- Port: 3000

## Health Check
Binary version check every 10s.
